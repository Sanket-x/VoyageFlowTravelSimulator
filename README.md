# VoyageFlow — Travel Simulator

This is a basic frontend project built using HTML, CSS, and JavaScript.

The idea behind this project was to create a simple travel website where a user can go through a complete flow — entering details, exploring destinations, selecting options, and seeing a final summary.

## Features

* User registration with basic validation
* Destination browsing section
* Travel option selection (flight, food, activities)
* Dynamic price calculation
* Final booking summary

## Tech Stack

* HTML
* CSS
* JavaScript

## Purpose

This project was built to practice core frontend concepts like form handling, DOM manipulation, and building a multi-step user flow.

## Note

This is a simulation project. No real booking or payment is involved.
